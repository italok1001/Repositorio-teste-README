 ##📋 Estrutura de um README

Um README geralmente contém:

```markdown
# Nome do Projeto

Descrição do projeto.

## Funcionalidades

- Funcionalidade 1
- Funcionalidade 2

## Tecnologias

- HTML
- CSS
- JavaScript

## Como executar

Passo a passo para rodar o projeto.

## Autor

Seu nome
```

---

✍️ Principais comandos Markdown

### Títulos

```markdown
# Título 1
## Título 2
### Título 3
```

Resultado:

# Título 1
## Título 2
### Título 3

---

### Texto em destaque

```markdown
**Negrito**

*Itálico*

~~Riscado~~
```

Resultado:

**Negrito**

*Itálico*

~~Riscado~~

---

 ##Listas

```markdown
- Item 1
- Item 2
- Item 3
```

ou

```markdown
1. Primeiro
2. Segundo
3. Terceiro
```

---

##Links

```markdown
[GitHub](https://github.com)
```

Resultado:

[GitHub](https://github.com)

---

##Imagens

```markdown
![Descrição da imagem](URL_DA_IMAGEM)
```

---

##Código

Código em linha:

```markdown
`console.log("Olá Mundo");`
```

Bloco de código:

````markdown
```javascript
console.log("Olá Mundo");
```
